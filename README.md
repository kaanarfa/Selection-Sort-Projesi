# Selection-Sort-Projesi
Bir diziyi nasıl sıralarız Ödev 1

## Proje 1

### Selection Sort Adımları
 [22,27,16,2,18,6] -> [2,27,16,22,18,6] -> [2,6,16,22,18,27] -> [2,6,16,18,22,27] 

### Big-O gösterimi 
O(n^2) = O(6^2) = O(36)

18 Sayısı dizinin ortasındadır.
